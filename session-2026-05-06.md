# Session Notes: May 6, 2026

## What We Built (All 6 Phases Complete)

### Phase 1: Foundation & Infrastructure
- **New URL params**: `parentfid`, `sortorderfid`, `wbsfid`, `deptypefid`, `durationfid`, `readonly`, `autosave`, `skipweekends`, `holidays`
- **Undo/Redo**: Ctrl+Z / Ctrl+Y, toolbar buttons with counts, 50-entry stack
- **Read-Only Mode**: `readonly=1` disables all edits, shows pill indicator
- **Manual Save Mode**: `autosave=0` queues changes locally, Save/Discard buttons with pending count badge

### Phase 2: Working-Day Calendar & Task Hierarchy
- **Working-day calendar**: skips weekends by default, `holidays` param for custom dates, shaded in timeline
- **Task hierarchy**: `parentfid` enables parent/child tree, expand/collapse chevrons, summary bracket bars for parents, rolled-up dates and percent

### Phase 3: Dependency System Upgrade
- **All 4 dependency types**: FS, SS, FF, SF with type-aware connection points, violation checking, and cascade logic
- **Drag to create dependencies**: connector circle on bar right edge (hover), drag SVG line to target bar, creates FS dep via API

### Phase 4: List Panel Enhancements
- **Inline cell editing**: double-click name or status cells to edit in place
- **Drag to reorder**: grip handle column, HTML5 drag-and-drop, updates sort order via API

### Phase 5: Timeline Bar Enhancements
- **Task labels toggle**: checkbox in filter bar to show/hide bar labels
- **Progress drag**: draggable handle on percent fill edge, saves percent via API

### Phase 6: Polish
- **localStorage persistence**: saves zoom, groupBy, toggles, collapsed state per project
- **Prev/Next navigation**: arrow buttons scroll timeline by zoom-appropriate days
- **CSV export**: toolbar button, downloads all filtered tasks

### Late Addition: Dependency Management in Quick Edit Panel
- Double-click a bar to open side panel
- Shows existing predecessors and successors by task name with type badge and lag
- Delete button (x) on each dep, calls API to remove record
- "Add Dependency" form: task name dropdown, direction picker (is predecessor / is successor), type (FS/SS/FF/SF), lag input
- `deleteDependency()` function with undo support

## Bugs Fixed
